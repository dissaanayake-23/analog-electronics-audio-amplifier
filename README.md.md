# Analog Electronics Audio Amplifier

## Project Overview
This project involves the design, construction, and testing of a multi-stage discrete audio output circuit for AM broadcast reception.

The circuit consists of:
- High-pass filter (1 kHz cutoff)
- Low-pass filter (20 kHz cutoff)
- Common-emitter BJT amplifier
- Audio output stage

## Objectives
- Filter the detected AM signal to the audio range (1–20 kHz)
- Amplify weak audio signals using a BJT amplifier
- Produce a clean and identifiable audio output
- Compare theoretical and practical results

## Components Used
- BC547 NPN Transistor
- Resistors
- Capacitors
- Breadboard / Dot Board
- DC Power Supply (12V)
- Oscilloscope
- Function Generator

## Results
- Measured Q-point: 6.21 V
- Measured Voltage Gain: 30.4
- Successfully filtered unwanted frequencies
- Produced a clean amplified audio signal
